# 📊 Stock Market Data Analyzer

## 📌 Overview

This project is a Stock Market Data Analyzer built using Python, Pandas, NumPy, and Jupyter Notebook. It focuses on analyzing historical stock data to extract meaningful insights such as trends, risk, returns, and relationships between multiple companies.

The project demonstrates real-world data analysis skills including data cleaning, time-series analysis, and financial insights generation.

---

## 🎯 Objectives

* Analyze stock price trends over time
* Calculate and visualize moving averages
* Measure daily returns and volatility (risk)
* Compare multiple companies
* Identify correlations between stocks

---

## 🛠️ Tech Stack

* Python
* Pandas – data manipulation
* NumPy – numerical computations
* Matplotlib– data visualization
* yFinance– stock data extraction
* Jupyter Notebook – analysis environment

---

## 📥 Data Source

Stock data is fetched using the Yahoo Finance API via the `yfinance` library.

Example stocks analyzed:

* Apple (AAPL)
* Google (GOOG)
* Tesla (TSLA)

---

## 📈 Features & Analysis

### 🔹 1. Stock Price Visualization

* Plotted closing prices over time
* Identified overall trends

### 🔹 2. Moving Averages

* 50-day moving average
* 200-day moving average
* Used to identify bullish/bearish trends

### 🔹 3. Daily Returns

* Calculated percentage change in stock prices
* Used to understand short-term performance

### 🔹 4. Risk (Volatility) Analysis

* Measured standard deviation of returns
* Higher value indicates higher risk

### 🔹 5. Multi-Stock Comparison

* Compared performance of multiple companies
* Visualized returns on the same graph

### 🔹 6. Correlation Analysis

* Identified relationships between stocks
* Helps understand market behavior

---

## 🧠 Key Insights

* Different stocks exhibit different risk levels
* Moving averages help identify long-term trends
* High volatility indicates higher investment risk
* Some stocks show strong correlation, meaning they move together

## 🚀 How to Run the Project

1. Clone this repository:

```
git clone https://github.com/your-username/stock-market-analyzer.git
```

2. Install required libraries:

```
pip install pandas numpy matplotlib yfinance notebook
```

3. Open Jupyter Notebook:

```
jupyter notebook
```

4. Run all cells in:

```
stock_market_analysis.ipynb
```
